### **ÉTAPE 1️⃣: Générer SECRET_KEY (1 min)**
```bash

```
Copie la clé générée

### **ÉTAPE 2️⃣: Push le code (1 min)**
```bash
git add .
git commit -m "🚀 Optimizations"
git push origin main
```

### **ÉTAPE 3️⃣: Deploy sur Render (3 min)**
1. render.com → Create Web Service
2. Select ton repo
3. Add env variables (SECRET_KEY, DATABASE_URL, etc)
4. Click Deploy

## 🆘 **TROUBLESHOOTING RAPIDE**

| Problème | Solution |
|----------|----------|
| "Static files 404" | `python manage.py collectstatic --noinput` |
| "Database error" | Check `DATABASE_URL` in Render settings |
| "Images not loading" | Check Cloudinary credentials |
| "Slow page load" | Check Render logs, verify caching |
| "CSRF error" | Add domain to `CSRF_TRUSTED_ORIGINS` |
| "Admin not accessible" | Check login credentials |
